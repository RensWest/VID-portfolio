Interface & Interactie

Sanne ‘t Hooft

Bij Interface & Interactie kon je kiezen uit twee opdrachten. Ontwerp en realiseer één pagina - met vanilla html, css en indien nodig js. Ik heb gekozen voor opdracht 1: een aantrekkelijke, informatieve en passende tijdlijn voor een inmiddels klassiek karakter/product. Hierbij heb ik het karakter Mario Gekozen.

Naar GitHub Procesverslag Naar Prototype

Bij deze opdracht heb ik super Mario door de jaren heen gemaakt. Je kan door de tijd heen door op volgende en vorige te klikken. Ik heb er leuke elementen van Mario in verwerkt en alle vormgeving met CSS te maken (zoals de groene buizen links en rechts).

Ik heb geleerd dat ik eerst een plan moet maken met waar welke elementen moeten komen voordat ik begin met coderen. Nu was ik gewoon maar begonnen met een basis idee en ben ik daar verder op gaan bouwen met nieuwe ideetjes.

Beginnen met Javascript ging voor mijn niveau wel goed. Ik wist de basis op te zetten voor de functies die ik uit wilde gaan voeren. Alleen de complexere dingen had ik hulp bij nodig, maar begrijp ik wel en dacht ik bij mee tijdens de hulp. Ik heb zelfs zelfstandig Javascript code gemaakt die ik voorheen niet zou kunnen.
